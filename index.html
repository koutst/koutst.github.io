Flight Traffic

Flight Traffic is a flight mapping application built in D3.js. The map was built using TopoJSON, an extension of GeoJSON, using coordinates in order to build each individual country. Major airports are rendered on top of that map. Flights are rendered on top of that. Because of the enormous number of daily flights around the world, Flight Traffic maps domestic flights in the US for January 1st 2017
